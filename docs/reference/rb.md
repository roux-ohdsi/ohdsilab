# Shortcut to rollback in the case of a redshift error

Shortcut to rollback in the case of a redshift error

## Usage

``` r
rb(con = getOption("con.default.value"))
```

## Arguments

- con:

  defaults to the default connection in the session. either set using
  options or supplied directly

## Value

nothing

## Examples

``` r
if (FALSE) rb() # \dontrun{}
```
